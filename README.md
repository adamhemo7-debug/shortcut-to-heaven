# Shortcut to Heaven ✨

A beautiful Islamic daily dhikr (remembrance) companion app to help you stay consistent with Islamic practices.

## 🌟 Features
- **Daily 7 Athkar** - Essential daily remembrances
- **Morning & Evening Athkar** - Daily protection
- **99 Names of Allah** - Learn and contemplate
- **Progress Tracker** - Track your consistency and build streaks
- **Offline** - Works completely offline
- **Beautiful Design** - Dark theme with gold accents
